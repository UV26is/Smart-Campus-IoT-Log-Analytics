# Mini Project 2 Code and Outputs

This folder contains the main code files and generated results for the smart-campus IoT log analytics project.

## Folder Structure

- `mapreduce/`: Python mapper and reducer scripts used for Hadoop Streaming.
- `ray/`: Ray-based abnormal device detection program.
- `outputs_hadoop/`: Output files generated by the Hadoop MapReduce jobs.
- `outputs_ray/`: Output files generated by the Ray program.

## Main Files

- `mapreduce/mr_mapper.py`: Mapper script for processing IoT log records.
- `mapreduce/mr_reducer.py`: Reducer script for aggregating MapReduce results.
- `ray/abnormal_detection_ray.py`: Ray program for detecting abnormal devices.

## Output Files

The Hadoop output folder contains:

- `sensor_count.txt`
- `warning_error_by_building.txt`
- `top10_devices.txt`

The Ray output folder contains:

- `abnormal_devices.csv`
- `abnormal_devices_unique.csv`
- `time_ray.txt`

These files were generated from the same IoT log dataset used in the project pipeline.
